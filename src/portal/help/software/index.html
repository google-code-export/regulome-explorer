About Our Analytical and Exploratory Software

All software created at CSACR is free and open source. Current projects are maintained through public social code repositories.

Google Code

If you wish to use, modify, or contribute to our software, you can find a list of our projects on Google Code hosted under the codefor@systemsbiology.org username. Please feel free to submit issues, bugs, etc to the corresponding project's repository (like Regulome Explorer).

GitHub

Projects can also be found on GitHub. Most are gathered under the codeforsystemsbiology organization.

Questions, concerns, or comments can be directed to the listed project owner or codefor@systemsbiology.org.
